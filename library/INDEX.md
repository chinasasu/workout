# Library Index

This repo is organized for low-token retrieval.

## Read Order

1. `README.md`
2. `library/INDEX.md`
3. `library/catalog.md`
4. One muscle-group file, if needed

## Fast Lookup

- Action name -> `library/catalog.md`
- Back exercises -> `library/back.md`
- Legs exercises -> `library/legs.md`
- Chest exercises -> `library/chest.md`
- Shoulder exercises -> `library/shoulders.md`
- Arm exercises -> `library/arms.md`
- Core exercises -> `library/core.md`
- Rehab / pain-management items -> `library/rehab.md`

## Entry Shape

Each exercise entry uses the same fields:

- Exercise name
- Video link
- Target muscles
- When to use it

## Retrieval Rule

If the user asks for a recommendation, start from `library/catalog.md`, then open only the relevant muscle-group file for supporting context.
